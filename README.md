## Veramo Node Tutorial

Example repo for Veramo Node Tutorial: [https://veramo.io/docs/node_tutorials/node_setup_identifiers](https://veramo.io/docs/node_tutorials/node_setup_identifiers)

This code works with Node.js 18.* and above.

## Usage

    yarn
    yarn node --loader ts-node/esm ./src/list-identifiers.ts
    yarn node --loader ts-node/esm ./src/create-identifier.ts
    yarn node --loader ts-node/esm ./src/create-credential.ts
    yarn node --loader ts-node/esm ./src/verify-credential.ts
